🔎 Énigme 2 : Que fait ce code Python ?

from nltk import FreqDist
text = ['the', 'sea', 'the', 'whale', 'the', 'ship']
FreqDist(text).most_common(1)

À quoi sert ce code ?

👉 Il affiche le mot le plus fréquent dans la liste : ('the', 3).
